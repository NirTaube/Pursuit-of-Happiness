API Examples

Get Total Net Migration for a Country

Retrieve the total net migration for a specific country.

Endpoint:

/migration/total/COUNTRY

Example:

/migration/total/United States

Get Top 25 Countries for a Year

Retrieve the top 25 countries based on net migration for a specific year.

Endpoint:

/migration/top/YEAR

Example:

/migration/top/2022

Get Bottom 25 Countries for a Year

Retrieve the bottom 25 countries based on net migration for a specific year.

Endpoint:

/migration/bottom/YEAR

Example:

/migration/bottom/2022

Get Top 15 Countries with Highest Percentage Increase in Net Migration

Retrieve the top 15 countries with the highest percentage increase in net migration over a specified year range.

Endpoint:

/migration/percentage_increase?year_range=START_YEAR-END_YEAR

Example:

/migration/percentage_increase?year_range=2018-2022

Get Average Net Migration for a Country

Retrieve the average net migration for a specific country.

Endpoint:

/migration/average/COUNTRY

Example:

/migration/average/Japan

Get Yearly Change in Net Migration for a Country

Retrieve the yearly change in net migration for a specific country.

Endpoint:

/migration/yearly_change/COUNTRY

Example:

/migration/yearly_change/Israel

Get Happiness Score for a Country

Retrieve the happiness score for a specific country.

Endpoint:

/happiness_score/COUNTRY

Example:

/happiness_score/North Korea

Get Country Factor Value

Retrieve the value of a specific factor for a country.

Endpoint:

/country_factors/COUNTRY/FACTOR

Example:

/country_factors/Turkey/business
